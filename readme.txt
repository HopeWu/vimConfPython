A vim configration file for writing python

To Install this, you need to do:
Step1. Install vundle: 
  # git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim
Step2. Copy the .vimrc file in the downloaded directory to ~
Step3. Open .vimrc with vim and run this command: PluginInstall
Done!
